Box Model Example

This is a simple example of the Box Model. In this example, there are three different HTML divisions (Title, Descprition, Surrounding Box) formatted in CSS to take the form of the Box Model. Simply put, the box model consists of margins, padding, borders, and content. The "title" division and the division that this descpription is in are formated to match the width of the embeded Youtube video. The "formula" to match the width of any item is left padding + right padding + left border + right border + left and right margins.
